# NOVA Studio

A modern Design Agency Homepage built using React.js, Tailwind CSS, and Framer Motion.

## Features

- Modern UI/UX Design
- Responsive Layout
- Smooth Scroll Navigation
- Framer Motion Animations
- Portfolio Showcase
- Contact Section
- SEO Optimized
- Mobile Friendly

## Tech Stack

- React.js
- Tailwind CSS
- Framer Motion
- Lucide React
- React Icons

## Installation

```bash
npm install
npm run dev
